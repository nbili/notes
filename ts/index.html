# TypeScript 是JavaScript类型的超集，可以编译成纯JavaScript

TypeScript 提供了强大的类型系统，可以在开发阶段便发现许多问题。

TypeScript 的编译过程大概如下：

  1. TypeScript 源码 --> TypeScript AST
  2. AST 类型检查
  3. TypeScript AST --> JavaScript 源代码
